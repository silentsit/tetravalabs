# v2-photos-white

Canonical white-background product PNG archive for the storefront.

- **Deploy path:** `apps/storefront/public/products/v2/` (what the live site serves)
- **Sync:** `npm run sync:v2-images` (copies PNGs from here into the deploy path)
- **Map handles:** `npm run map:images`

Do not delete this folder. If it is missing, restore PNGs from `apps/storefront/public/products/v2/` and re-run sync when updating assets.
